# DevOps URL Shortener

A Flask-based URL Shortener application built to learn and demonstrate real-world DevOps practices.

## Tech Stack

- Python
- Flask
- Docker
- Docker Compose
- Kubernetes

## Project Structure

```
devops-url-shortener/
├── app.py
├── database.py
├── Dockerfile
├── docker-compose.yml
├── requirements.txt
├── k8s/
└── README.md
```
